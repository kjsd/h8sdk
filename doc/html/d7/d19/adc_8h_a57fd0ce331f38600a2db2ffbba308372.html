void ADC_enable ( ADC_Channel  ch)

チャネル有効化

使用するチャネルを設定する。ノーマルモード時は指定チャネルのみ。ス キャンモード時は指定チャネルより若い番号のチャネルは全て使用可能に セットされる。例えば、ADC_AN2 を与えた場合 AN0〜AN2のチャネルが有効 になる。

引数
[in]ch有効にするチャネル
