void SOUND_setTone ( _UBYTE  idx,
const SOUND_Tone *  tone 
)

音登録

音を登録する。SOUND_MAX_PRONOUNCE個のバンクに同時にセットすることができる。 いつでも設定可能。

引数
[in]idxバンクのインデックス。0〜SOUND_MAX_PRONOUNCE-1
[in]tone設定するSOUND_Toneへのポインタ
無音(SOUND_Tone::freq == 0)、またはSOUND_Tone::evlpがNULLのデータを 登録することはできない。
